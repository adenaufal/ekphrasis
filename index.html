Templates, placeholders, composition presets, and batch queue
— injected directly into NovelAI.

↓ Install Script

Requires Tampermonkey (Chrome / Firefox / Edge / Safari). Click install, confirm, then open novelai.net/image.

Templates
Save and reuse prompt snippets. Organize by category. One click to apply.
Placeholders
Define {artist}, {character}, and custom types. Resolved at queue time.
Batch Queue
Auto-generate all permutations of templates × placeholders with a configurable delay.
Framing Presets
One-click shot distance, camera angle, body focus, and pose tags — straight into the prompt.
Randomizer
Use ||opt1|opt2|| syntax to expand variants into separate queue entries automatically.
Weight Syntax
Supports 1.5::tag:: numerical weights (V4.5+) and legacy {tag} / [tag] multipliers.
